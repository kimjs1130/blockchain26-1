## How the Website Works (홈페이지 동작 원리)

이 홈페이지는 CoinGecko Public API를 사용하여
Bitcoin과 Ethereum의 가격을 실시간으로 표시합니다.
홈페이지 주소: https://kimjs1130.github.io/blockchain26-1/
### 1️⃣ 초기 실행

웹페이지가 열리면 JavaScript 함수 `fetchPrices()`가 즉시 실행됩니다.

이 함수는 CoinGecko API에 요청을 보내고,
받아온 JSON 데이터를 화면에 표시합니다.

---

### 2️⃣ 자동 업데이트 구조

이 홈페이지는 `setTimeout()`을 사용하여
일정 시간 간격마다 자동으로 가격을 다시 가져옵니다.

기본 업데이트 간격은 **10초**입니다.

---

### 3️⃣ API 오류 대응 방식 (최대 2분 증가 구조)

만약 API 요청이 실패하거나 (예: Too Many Requests, 네트워크 오류 등),
서버에서 정상적인 응답을 받지 못하면:

- 요청 간격이 2배로 증가합니다.
- 예: 10초 → 20초 → 40초 → 80초
- 최대 120초(2분)까지 증가합니다.

이 방식은 CoinGecko의 요청 제한(rate limit)을 피하기 위한 안전 장치입니다.

성공적으로 데이터를 받아오면,
업데이트 간격은 다시 10초로 복구됩니다.

---

### 4️⃣ 실시간 표시 방식

- 가격은 페이지를 새로고침하지 않고 자동으로 변경됩니다.
- 마지막 업데이트 시간(`Last updated`)도 함께 갱신됩니다.
- 기존 가격은 유지되며, 오류 발생 시 화면이 멈추지 않도록 설계되었습니다.

---

이 구조를 통해 안정적인 실시간 가격 표시가 가능하도록 구현하였습니다.
